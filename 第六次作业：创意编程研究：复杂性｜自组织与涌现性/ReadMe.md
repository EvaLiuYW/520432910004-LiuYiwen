程序说明：
-------

### 设计意图：

  > 本程序意在模拟一些微观生物的避光行为。
  > 
  > 当生物所在位置被照亮时，它们便会自发地向四周发散。
  > 
  > 生物碰壁时，伴有能量损耗，运动速度减缓直至为0.


### 程序逻辑：

  > 运用向量变量模拟排斥力行为，力的大小和距离大小成反比关系。


### 实现效果：
  
  > 鼠标指针代表光源中心。照亮区域半径为200。
  > 
  > 圆点模拟微观粒子，当圆点处于以指针为圆心，200为半径的圆形区域内时，它们会受到斥力作用向四周发散。
  > 
  > 按“r”键重置画面。
