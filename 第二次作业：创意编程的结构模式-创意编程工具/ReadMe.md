# 【第二次作业】作业说明
本次作业共含有三个子作业：

## 1. evaliu_lines2
本作业意在运用random函数，随机创建不同位置、不同粗细、不同颜色的线条。

黑色底色配合高明度低彩度的颜色，达到视觉上“发光”的效果，模拟了霓虹灯管的光效。

## 2. evaliu_masaike
本作业运用random函数，配合鼠标的交互功能，表达了“马赛克笔刷”的意味。

### - 实现效果
当鼠标左键被按下时，画布上就会随机在光标的邻近位置处生成大小各异、颜色各异的正方形，
